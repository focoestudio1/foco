⚙ Conectar con Google Sheets

Sigue estos pasos una sola vez. Después la app funcionará igual en cualquier dispositivo.

Paso 1 — Crear la hoja de cálculo

Abre tu Drive en la carpeta FOCO: Abrir carpeta
Crea una nueva Hoja de Cálculo de Google llamada FOCO_Base_de_Datos.

Paso 2 — Abrir el editor de Apps Script

Dentro de la hoja de cálculo, ve a menú Extensiones → Apps Script.
Borra todo el código que aparece y pega el código de abajo.

Paso 3 — Obtener el ID de tu hoja

En la URL de tu hoja de cálculo aparece así:
https://docs.google.com/spreadsheets/d/ESTE_ES_EL_ID/edit
Copia ese ID y reemplaza TU_SPREADSHEET_ID en el código.

Paso 4 — Publicar como aplicación web

En Apps Script: Implementar → Nueva implementación → Aplicación web
• Ejecutar como: Yo (tu cuenta Google)
• Quién tiene acceso: Cualquier usuario
Copia la URL de la aplicación web que aparece.
